# Sourdough Buddy

Your sourdough baking companion — track starters, calculate recipes, time your bakes, and log results.

## Features

- **Dashboard** — At-a-glance view of starter health, active bakes, and recent logs
- **Starter Tracker** — Log feedings with flour type, ratios, and notes. Monitor starter health over time
- **Recipe Calculator** — Enter target dough weight, hydration %, and get exact ingredient amounts with baker's tips
- **Bake Timer** — Step-by-step guided timers with preset recipes (Country Loaf, Same-Day Loaf)
- **Bake Log** — Journal each bake with photos, ratings (crumb, taste, rise), notes, and "what I'd change"

## Getting Started

```bash
npm install
npx expo start
```

Scan the QR code with Expo Go (Android) or Camera app (iOS).

## Tech Stack

- Expo SDK 55 + expo-router (file-based routing)
- React Native with TypeScript
- AsyncStorage for local persistence
- Warm earth-tone design system
